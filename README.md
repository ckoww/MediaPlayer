# MediaPlayer
簡易多媒體播放器，透過內建Windows Media Player控制項(WMP)播放影片與音訊檔案。
使用步驟:選擇影片 → 載入 → 播放 → 暫停 /停止
<img width="651" height="449" alt="image" src="https://github.com/user-attachments/assets/6fba85f5-c9bc-4d27-9425-3b02e935b6c8" />
支援常見影音格式，例如：.wmv、.mp4、.avi、以及其他系統可播放的格式
1. 選擇媒體檔案（Browse）
開啟檔案選擇視窗，支援多種影片格式（WMV/MP4/AVI/All Files）
選擇後自動載入至播放器（不自動播放）
2. 播放（Play）
播放已載入的影片
為避免錯誤操作，加入播放前檢查(如下)
<img width="133" height="133" alt="image" src="https://github.com/user-attachments/assets/b9b3c60d-e8ad-4c2a-91f7-397459d4dfcd" />
3. 暫停（Pause）
暫停目前播放畫面
4. 停止（Stop）
停止播放並回到初始狀態
